## Aeris AI

Aeris AI is a Java-based virtual assistant.  
It processes user commands and performs backend-driven tasks.

## Features
- Command-based interaction
- Backend logic implementation
- Modular project structure
- Secure configuration management

## Built With
- Java 17
- Maven
- Spring Boot

## Security
Sensitive configuration files like `application.properties` are excluded using `.gitignore`.

## 📦 How to Run

1. Clone the repository:
git clone https://github.com/yourusername/aeris-ai.git

2. Navigate into the project folder:
cd aeris

3. Add your own `application.properties` file inside:
src/main/resources/

4. Run the project:
mvn spring-boot:run

5. Open your browser and visit:
http://localhost:8080/aeris.html
